## Project Overview

The Student Fee Portal is a dashboard-like application built with React + Vite.
It enables students to:
1) View overall school fees
2) Track payments
3) Download receipts
4) Navigate through clean UI sections

This project demonstrates your frontend skills in UI layout, state management, and component organization.
