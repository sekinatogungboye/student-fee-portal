## Project Overview
The **Student Fee Portal** is a React-based web application that allows students to track and manage their school fees efficiently.  
Users can:  
- View fee summaries   
- See transaction history  

**Live Demo:** [View Live App](https://app.netlify.com/projects/sekinatfinalproject/overview)

---

## Features
- **Responsive card-based dashboard**  
- Highlight **unpaid fees**  
- **Chart.js visualization** (planned/future)  
- Transaction history stored in **LocalStorage**   

---

## Tools & Technologies
- **Frontend Framework:** React (Vite)  
- **Styling:** Tailwind CSS (CDN)  
- **Programming:** JavaScript (ES6)  
- **Libraries:** html2pdf.js, Chart.js (planned)  
- **Persistence:** LocalStorage
